# EntokMart - E-Commerce Marketplace Entok

Marketplace untuk menghubungkan peternak entok dengan pembeli.

## Setup

```bash
# Create virtual environment
python3 -m venv venv
source venv/bin/activate

# Install dependencies
pip install -r requirements.txt

# Run
python app.py
```

Akses: http://127.0.0.1:5000

## Akun Demo

 Buat manual melalui register page.

## Environment Variables

Copy `.env.example` ke `.env` dan isi SECRET_KEY untuk production.
